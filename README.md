# 🚀 Prepify AI — AI-Powered Job Preparation Platform

Prepify AI is a full-stack AI-driven web application designed to help users prepare for interviews through resume analysis, skill gap detection, semantic search, and AI-generated interview preparation.

The platform uses Retrieval-Augmented Generation (RAG) with transformer embeddings and Groq LLMs to generate personalized interview guidance for both technical and non-technical roles.

---

## ✨ Features

- 🔐 JWT Authentication & Secure Sessions
- 🤖 AI-Powered Interview Question Generation
- 📄 Resume Parsing & ATS Resume Generation
- 🧠 Skill Gap Detection & Match Score Analysis
- 📊 Personalized Learning Roadmaps
- 🔎 Semantic Search using Transformer Embeddings
- ⚡ Retrieval-Augmented Generation (RAG) Pipeline
- 📥 PDF Resume Generation & Download

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Manan03-o-o/-Job-Preparation-Web-Application.git

cd -Job-Preparation-Web-Application
```

### Install Dependencies

#### Backend

```bash
cd Backend
npm install
```

#### Frontend

```bash
cd Frontend
npm install
```

---

## 🔑 Environment Variables

Create a `.env` file inside Backend:

```env
PORT=3000
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret
GROQ_API_KEY=your_groq_api_key
```

---

## ▶️ Run Project

### Start Backend

```bash
cd Backend
npm run dev
```

### Start Frontend

```bash
cd Frontend
npm run dev
```
